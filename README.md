# AntColonyOptimization-ACO-
Le projet Ant Colony Optimization a été lancé par mon ami en prépa. Il simule une colonie de fourmis pour trouver le chemin le plus court grâce aux phéromones, avec une application possible en milieu urbain. En BUT R&amp;T 2e année, je veux l’aider pour mettre mes compétences en pratique et découvrir l’ACO.
